# 🛍️ Nostra - E-commerce Website Clone

Nostra is a modern and responsive **e-commerce website clone** built using HTML, CSS, and JavaScript.
It showcases a stylish UI with product sections, interactive elements, and a clean user experience inspired by real-world online stores.

---

## 🚀 Features

* 🔥 Attractive homepage with hero slider
* 🛒 Product sections: New Arrivals & Most Wanted
* 💎 Highlighted services (Best Products, Secure Shopping, etc.)
* 🎯 Limited-time offer section
* 📱 Fully responsive design
* 🍔 Mobile-friendly side navigation menu
* ❤️ Interactive UI elements (like buttons, sliders, overlays)
* 📧 Newsletter subscription section
* 🧾 Structured footer with multiple links

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling & Layout
* **JavaScript ( JS)** – Interactivity
* **Font Awesome** – Icons
* **Google Fonts (Poppins)** – Typography
